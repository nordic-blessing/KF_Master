# 2026赛季 R2自动控制部分（STM32F446RCT6）
# KF_Master

## 代码逻辑
此项目主要负责R2全自动控制，主要完成的任务是武器端头的夹取和对接以及KFS的抓取和放置。  
代码基于 _FreeRTOS_ ，使用 _CMSIS-RTOS_ 接口。项目中共创建了2个任务和1个事件组，下面将详细介绍每一部分的具体功能：
### 任务
- **InitTask:** 任务开始前的全局初始化
- **CmdTask:** 任务的全局发送命令，以时间顺序进行动作命令的发布，通过各机构的完成通知实现任务的推进。

### 事件组
- **KFQEvent:** 标记底盘和机构的动作，用于等待底盘或机构完成指定动作  

## 文件结构
**Application-任务函数**
- CommandTask: 命令反馈（将FDCAN发送函数封装至应用层, 完成指定动作后向主控发送完成通知）
- CtlCmd: 全局控制命令发布，以及所需动作的定义和实现（根据主控的命令进入指定的动作函数）
- MC: 武馆
- MF: 梅林
- Arena: 对抗区
- MFTask: 串口接收MF_Cheat的数据后，对规划路径进行解析

**SPLIB-接口库**
- 包含了所需的板级支持文件、设备驱动、常用控制函数等。此处仅对本项目中使用的部分做说明，具体说明见SPLIB的说明文档
- Bsp/bsp_can: 封装过滤器的配置使能、接收中断函数的配置以及发送函数
- Bsp/bsp_uart: 使能串口接收中断，并配置回调函数
- Device/Commander/commander_can: 此处将CAN的发送和接收进一步做应用层的封装，专用于板件通信
- Devices/Visual/visual_uart: 用于与视觉串口通信

**User-自定义函数**
- inc_user：全局头文件，事件组标志位的定义
- init_user：全局初始化部分，以及部分回调函数的具体实现
